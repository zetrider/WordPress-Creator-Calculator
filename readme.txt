WPCC - creates forms calculators with the construction of the formula for the calculation, and further results sent to email.

== Description ==

<strong>EN: </strong><br>

<strong> Possible to add fields to the constructor : </strong> <br>
* Unlimited calculators <br>
* Field : Text to enter a description <br>
* Field : SELECT dropdown <br>
* Field : Checkbox multiple choice <br>
* Field : Radio single choice <br>
* Field : Input Text <br>
* Field : Date <br>
* Field : Input Hidden <br>
* Field : $_SESSION <br>
* Field : jQuery copies data <br>
* Field : Slider UI scrolling <br>
* Field : Conditions for fields with the opportunity to ask operators : == , <, < =,> ,> =, between <br>
* Field : Conditions Hidden - return result depending on the value <br>
* Field : Arithmetic functions : square ; degree ; round <br>
* Field : Result Fields: result between fields <br>
* Field: Custom Fields: single custom fields <br>

<strong> Send results to E-Mail: </strong> <br>
* Setting: Enable or disable sending form <br>
* Setting: Subject of the letter <br>
* Setting: e-mail <br>
* Setting: form validation <br>
* Setting: clearance letters <br>

<strong> Results of calculation: </strong> <br>
* Setting: Add more than one result <br>
* Setting: Apply to the result of an arithmetic expression <br>
* Setting: Rounding <br>
* Setting: Making result <br>

<strong> Other settings: </strong> <br>
* Export and import of formula <br>
* Change the appearance of the calculator. The choice of four themes <br>
* Hook 'wpcc_result_filter' on the result

More information about the plugin <a href="http://zetrider.ru/" target="_blank">here</a>
<br><br>

<strong>RU:</strong><br>

<strong> Поля в конструкторе калькулятора : </strong> <br>
* Количество калькуляторов не ограничено <br>

* Поле : Checkbox множественный выбор <br>
* Поле : Radio одиночный выбор <br>
* Поле : Input Text для ввода данных <br>
* Поле : Даты <br>
* Поле : Input Hidden скрытое поле<br>
* Поле : $_SESSION для работы с сессиями<br>
* Поле : jQuery копируем данные<br>
* Поле : Slider UI слайдер <br>
* Поле : Conditions условие работы полей с возможностью задать оператор : == , <, < =,> ,> =, between <br>
* Поле : Conditions Hidden - скрытое условие для получения значения в зависимости от результата<br>

<strong> Отправка результатов на E-Mail: </strong> <br>

* Настройка: Указать на какой e-mail<br>
* Настройка: Валидация полей <br>

<strong> Результаты расчета: </strong> <br>

* Настройка: Позволяет добавить арифметическое выражение для каждого из результата <br>
* Настройка: Округление результатов <br>
* Настройка: Оформление результатов<br>

<strong> Другие настройки: </strong> <br>

== Installation ==

1. Download the plugin wp-creator-calculator.
2. Uploads folder wp-creator-calculator plug-in category /wp-content/plugins/
3. Activate the plugin
4. Select the admin panel WP Calculator
5. Customize to suit your needs
